# OSRS Routebound

A local-first companion app for the **Routebound** Old School RuneScape restrictive playthrough concept.

Routebound makes travel routes the legal boundary of the account. Pick or roll a route, then only train, gather, fight, shop, quest, or travel through content attached to that active route until every checkpoint is cleared.

## First Prototype

- Create a run and choose an active route.
- Roll curated sample routes by route type.
- Track checkpoints as reached and cleared.
- See route permissions for regions, skills, enemies, shops, transport, and quests.
- Keep breach notes and route history.
- Complete routes into an archive.
- Export and import local save files.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The app is a static Vite build and can be deployed to GitHub Pages.
